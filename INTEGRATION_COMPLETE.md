## 🧪 Комплексное тестирование

### Сценарий 1: Сотрудник создаёт заявку

```
1. Сотрудник: "Хочу отпуск с 15 по 25 марта"
   ↓
2. AI проверяет конфликты через API
   ↓
3. Если конфликтов нет:
   - AI генерирует <ACTION>
   - ChatWidget создаёт заявку
   - Сотрудник получает авто-ответ
   ↓
4. Если есть конфликты:
   - AI показывает карточку с деталями
   - Предлагает альтернативные даты
   - Кнопки для быстрого выбора
```

### Сценарий 2: Просмотр статистики

```
1. Откройте /dashboard
   ↓
2. Найдите "📊 Персональная статистика"
   ↓
3. Проверьте:
   - Прогресс бар
   - Балансы отпусков
   - Burnout Prevention
```

### Сценарий 3: День рождения

```
1. Convex Dashboard → Functions → birthdays
   ↓
2. Запустите: checkBirthdaysToday
   ↓
3. Проверьте уведомления:
   - Коллеги: "🎉 Сегодня день рождения у Ивана!"
   - Именинник: "🎂 С Днём Рождения!"
```
